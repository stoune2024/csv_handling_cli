# YouTube Stats CLI

CLI-приложение для анализа CSV-файлов с метриками YouTube-видео и формирования отчётов по ним.

---

## Поддерживаемые отчёты

### `clickbait`

Показывает видео, которые считаются кликбейтными по следующим условиям:

* `ctr > 15`
* `retention_rate < 40`

---

## Пример запуска

```bash
python -m app.cli --files stats1.csv stats2.csv --report clickbait
```

---

## Пример вывода

```text
| title                            |   ctr |   retention_rate |
|----------------------------------|-------|------------------|
| Секрет который скрывают тимлиды  |  25.0 |               22 |
| Как я спал по 4 часа             |  22.5 |               28 |
| Купил джуну макбук и он уволился |  19.0 |               38 |
```

---

## Запуск тестов

```bash
pytest
```

---

## Просмотр покрытия тестами

```bash
pytest --cov=app --cov-report=term-missing --cov-report=html
```

Команда использует плагин **pytest-cov** и:

* выводит процент покрытия тестами в консоль;
* показывает непокрытые строки (`term-missing`);
* создаёт HTML-отчёт с детальной статистикой покрытия в директории `htmlcov/` в корне проекта.

---

## Архитектура проекта

Проект спроектирован с учётом дальнейшего расширения:

* каждый отчёт реализуется отдельным классом;
* отчёты регистрируются через registry-декоратор;
* для добавления нового отчёта достаточно создать новый класс и зарегистрировать его;
* CLI, чтение CSV, логика отчётов и форматирование вывода разделены по слоям.

## Обработка ошибок

Приложение корректно обрабатывает:

* несуществующие файлы;
* передачу директории вместо файла;
* неизвестный тип отчёта;
* невалидные данные в CSV.

---

## Примечание

Содержимое CSV предполагается валидным по структуре согласно условиям задания,
но приложение дополнительно валидирует данные на уровне моделей.
